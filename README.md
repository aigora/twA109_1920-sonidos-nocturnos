# Título del trabajo

Analizador de sonidos nocturnos.

## Integrantes del equipo

Manuel Segade Bodelo (manuelsegadebodelo), Sofia Oliveira (sofiaoliveira1), Jorge Pacio Míguez (jotapacio)
## Objetivos del trabajo

El objetivo del trabajo es crear un dispositivo que capte y analice los sonidos mientras uno duerme (ej. ronquidos). Posteriormente estableceremos una serie de parámetros de análisis de estos y los clasificaremos.
